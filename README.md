# PlacementPrep-Tracker-

A responsive web application that helps students track their placement preparation.

## Features
- DSA Progress Tracker
- Aptitude Tracker
- Resume Checklist
- Mock Interview Records
- Company-wise Preparation Roadmap

## Technologies
- HTML
- CSS
- JavaScript
- Local Storage
